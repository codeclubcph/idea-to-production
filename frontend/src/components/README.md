# Components

Place your React components here as you progress through the workshop.

## Suggested components (implement during Checkpoints 6 & 7)

| Component         | Description                                         | Checkpoint |
|-------------------|-----------------------------------------------------|------------|
| `TaskList.tsx`    | Renders a list of tasks fetched from the backend    | 6          |
| `TaskCard.tsx`    | Displays a single task (title, status, description) | 6          |
| `TaskForm.tsx`    | Form to create or update a task                     | 7          |
| `StatusBadge.tsx` | Colour-coded badge for TODO / IN_PROGRESS / DONE    | 6          |

## Tips

- Keep components small and focused on a single responsibility.
- Use the `Task` type from `@/types/task` for prop types.
- Use the API functions from `@/lib/api` to fetch and mutate data.
